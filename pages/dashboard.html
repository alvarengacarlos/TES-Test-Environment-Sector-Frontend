Tutorial

Preparando credenciais AWS

Crie um usuário AWS para o TES fazer o uso. Este usuário deve conter a permissão de 'AdministratorAccess'. Segue link da documentação.

Crie as credenciais de acesso para este usuário. Segue link da documentação.

Acesse a página de credenciais da AWS e insira suas credenciais criadas anteriormente.

Criado aplicações

Acesse a página de aplicações e crie uma nova aplicação fazendo o upload do seu código fonte no formato '.zip'.

Na página de aplicações ainda provisione a infraestrutura da sua aplicação.

Aguarde o processo de provisionamento terminar. É possível acompanhar este processo nesta própria tela através do campo 'Estado' ou pelo serviço 'CloudFormation' no seu console AWS.

Rodando sua aplicação

Quando o provisionamento terminar você poderá acessar a máquina virtual criada pelo serviço 'EC2 Instance' no seu console AWS.

Acesse sua máquina virtual.

Copie o código fonte com o comando disponível na página de aplicações.

Agora você poderá subir sua aplicação em containers utilizando o Docker.

A máquina virtual tem a porta 80 liberada. Então sua aplicação deverá ser exposta nesta porta ou ter algum mecanismo que faça o encaminhamento da porta 80 para a sua aplicação.

O IP público da máquina pode ser identificado através do console AWS.
